السبب الذي حددته:
─────────────────
❌ الأيقونات كانت ناقصة في manifest.json

الحل الذي طبقته:
────────────────
✅ أضفت أيقونات SVG مضمنة (temporary fix)
✅ الآن manifest يحتوي على أيقونات صالحة

═══════════════════════════════════════════════════════════════

⚡ جرّب الآن:

1. اضغط Ctrl+F5 (أعد تحميل قوي)

2. افتح DevTools (F12) → Application → Clear Storage

3. أعد تحميل (F5)

4. ابحث عن خيار التثبيت:
   
   • Android: اضغط ⋮ → "Install app"
   • iPhone: اضغط Share → "Add to Home Screen"
   • Windows: اضغط ⋯ → Apps → Install

═══════════════════════════════════════════════════════════════

✅ تأكد من أن:

□ DevTools → Application → Manifest
  يظهر بدون أخطاء (علامة ✓)

□ Service Workers
  مسجل ومفعل (Active and running)

□ Cache Storage
  يحتوي على "pos-system-v1"

═══════════════════════════════════════════════════════════════

🎨 اختياري: أضف أيقونات PNG أفضل

إذا أردت أيقونة أفضل (بدلاً من SVG):

1. افتح: http://localhost:3000/images/icon-generator.html

2. اضغط الأزرار الزرقاء لتحميل الأيقونات

3. احفظها في: public/images/

4. عدّل manifest.json لاستخدام المسارات الحقيقية
